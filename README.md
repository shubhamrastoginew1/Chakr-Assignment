## Tech Stack Used

### Setup Instructions

1. Clone the repository:

   ```bash
   git clone https://github.com/shubhamrastoginew1/Chakr-Assignment.git
   ```

2. Install dependencies:
      ```bash
      npm install
      ```
      
3. Start application :
      ```bash
      npm run dev
      ```
Then the application will be started on http://localhost:5173 (On your Local Machine).
